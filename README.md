# 🌐 React Demo Website

This is a simple demo website built using React to understand **functional components, file structure, and basic styling**.

---

## 🚀 Features

* Home page with 5 sections:

  * Header
  * Hero
  * About Us
  * Contact Us
  * Footer
* Reusable **Header** and **Footer** components
* Clean and simple UI design
* Organized folder structure

---

## 📁 Project Structure

```
src/
│── components/
│   ├── Header.jsx
│   ├── Footer.jsx
│
│── styles/
│   ├── Header.css
│   ├── Footer.css
│
│── App.jsx
│── App.css
│── index.css
```

---

## 🛠️ Technologies Used

* React (Functional Components)
* JSX
* CSS (Basic Styling)

---

## ⚙️ How to Run the Project

1. Install dependencies:

```
npm install
```

2. Start the development server:

```
npm run dev
```

3. Open in browser:

```
http://localhost:5173/
```

---

## 🎯 Learning Objectives

* Understand **React component structure**
* Learn **reusable components**
* Practice **JSX and styling**
* Work with **project folder organization**

## 📌 Notes

* Header and Footer are reusable components inside `components/`
* Their styles are inside `styles/`
* Other sections are written in `App.jsx`
